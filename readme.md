# Hello Pong for 2 players

Este es un proyecto simple de Pong creado con Godot Engine y GDScript. El objetivo de este proyecto es aprender los conceptos básicos del desarrollo de videojuegos con Godot.

## Cómo ejecutar el proyecto

1.  Abre el proyecto con Godot Engine.
2.  Presiona F5 para ejecutar la escena principal (`main.tscn`).

## Contexto

- **Motor:** Godot Engine (versión 4 o superior)
- **Lenguaje:** GDScript
- **Objetivo:** Crear un juego de Pong simple, paso a paso.
